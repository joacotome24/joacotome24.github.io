Course Project Little Lemon

SwiftUI Project


For the Little Lemon food-ordering app project, I used SwiftUI to create the mobile application's user interface. The app displayed a list of food items that were available to order, along with their prices, descriptions, and images.

To retrieve data for the app, I made API calls to an online source. The data was fetched in JSON format, which was then parsed and converted into a Swift-compatible format for use in the app.

I also used CoreData to store and manage the app's data locally. The data was stored in a persistent store, allowing users to access the information even when they were offline.

Additionally, the app allowed users to filter the list of items by category, such as appetizers, entrees, or desserts. The app also included a search feature, which allowed users to locate items by name.

Overall, this project required a combination of SwiftUI, CoreData, and API calls to create a user-friendly food-ordering app for the Little Lemon restaurant.

Course Project Reservations App

SwiftUI Project


The Reservations App project required the use of SwiftUI to develop a user-friendly reservations app for the Little Lemon restaurant. Through working on this project, I gained a deeper understanding of the SwiftUI framework and its capabilities for developing modern, dynamic user interfaces.

The app involved creating a view with the Little Lemon Restaurant logo and integrating it into the LocationsView. The MainView was also used to display the LocationsView and create a dynamic list of elements containing several restaurant locations.

To ensure data validation, I created a text field validation code and incorporated an alert to display errors in a variable. The reservation button was also programmed to run the form validation function when tapped, ensuring that user input was accurate and appropriate.

Overall, this project required a solid grasp of SwiftUI concepts and programming techniques, allowing me to create a professional-grade reservations app for the Little Lemon restaurant.

Course Project React Native

React Native Project


The Little Lemon Newsletter app is a React Native app that allows users to easily sign up for the Little Lemon newsletter, ensuring that they always receive the latest updates from the restaurant.

The app is simple and easy to use, with a visually appealing layout that includes core components such as Text, Image and View. It features two screens, designed with the user in mind. The first screen allows users to input their information, including their name and email address, into the TextInput component. The Pressable component allows them to submit their information and sign up for the newsletter.

Once the user submits their information, they are directed to the second screen, which confirms their subscription and displays a welcome message. The app uses React Navigation, which allows users to easily move between screens, ensuring a seamless user experience.

Overall, the Little Lemon Newsletter app is a straightforward solution for users who want to stay up-to-date with the latest news and updates from the restaurant. Its intuitive design and easy-to-use features make it a great addition to any food lover's app collection.

Course Project Clima App

Networking & CoreLocation


The Clima app's primary function is to provide users with current weather information for any city they search for, as well as their current location. The app's sleek and intuitive user interface is designed to display current temperature, weather condition, and location name.

The use of the delegate design pattern and Swift protocols allows for seamless communication between various app components, ensuring that data is retrieved and displayed accurately and efficiently. Networking and JSON parsing are used to retrieve weather data from the OpenWeatherAPI, which is then displayed in the app.

The use of Core Location allows the Clima app to obtain the user's current location and display the corresponding weather information. The app also allows users to search for the weather in any city they choose.

The dark mode feature allows users to switch to a darker color scheme for comfortable use in low-light environments. This feature enhances the user experience and adds an additional layer of usability to the app.

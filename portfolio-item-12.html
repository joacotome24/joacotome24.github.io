Course Project Split Bill App

UIKit Project


I had the opportunity to create a bill splitter calculator that takes into account the tip and the number of people splitting the bill. The project provided me with valuable insights into how to develop multi-screen apps with animated navigation, which is essential for creating user-friendly mobile applications.

One of the most important skills I learned during this project was how to use optional binding, optional chaining, and the nil coalescing operator, which allowed me to write more efficient and elegant code. By using these advanced programming techniques, I was able to create a high-quality, polished app that is both easy to use and visually appealing.

Overall, this project was an excellent opportunity to learn about iOS development, and I am proud to have added it to my portfolio. It is a testament to my skills and knowledge in programming and demonstrates my ability to create high-quality, user-friendly mobile applications that provide practical solutions to everyday problems.

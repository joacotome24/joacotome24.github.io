Personal Project MLRecognizer

CoreML Project


For this project, I used Swift and CoreML to create an image recognition app. I trained the app using the inception v3 mlmodel and implemented the functionality to take a photo with the phone's camera and analyze it using the CoreML model.

I was able to recognize a variety of objects and classify them accordingly. It was a challenging project, but I was excited to learn more about machine learning and its applications in mobile development.

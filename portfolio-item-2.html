Personal Project Chat AI

OpenAI API Project


For my personal project called ChatAI, I developed a chatbot using the OpenAI API to create a "ChatGPT" like chatbot that could answer anything. I built the app using UIKit and made API calls to the OpenAI API to retrieve data for the chatbot's responses.

As someone who is deeply interested in Artificial Intelligence (AI), I wanted to create a project that showcased the capabilities of AI in a user-friendly way. The app allowed users to ask any question and receive a response from the chatbot that was generated using the OpenAI API.

The app's user interface was created using UIKit and the API calls were made using Swift's built-in networking library, which allowed the app to retrieve data from the OpenAI API and display it in real-time.

Additionally, the app allowed users to filter the list of items by category, such as appetizers, entrees, or desserts. The app also included a search feature, which allowed users to locate items by name.

Overall, the ChatAI project was a great opportunity to explore my interest in AI and to create a practical application that showcased the capabilities of this exciting technology.

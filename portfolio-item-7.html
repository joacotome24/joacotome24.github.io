Course Project Orders App

Data in Swift Project


For the Orders App project, I successfully achieved the learning objectives of retrieving data from a web server with REST APIs, decoding JSON data into a suitable Swift model, and performing sorting and filtering operations to display retrieved data in an organized manner using CoreData.

The app allowed users to retrieve data from a web server using REST APIs, which was then decoded into a suitable Swift model. The data was organized into different categories and displayed in a user-friendly manner, allowing users to easily navigate and access the information they needed.

CoreData was also used to store and manage the app's data locally, allowing users to access the information even when they were offline. The app was designed to allow for easy sorting and filtering operations, making it easy for users to find the specific data they were looking for.

Overall, this project required a strong understanding of REST APIs, JSON data, and CoreData to successfully achieve the learning objectives.
